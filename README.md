# 📄 Smart Resume ATS Analyzer

An AI-powered Resume ATS Analyzer that helps job seekers optimize their resumes for Applicant Tracking Systems (ATS).

## ✨ Features

- 📤 Upload Resume (PDF/DOCX)
- 📊 ATS Compatibility Score
- 🔍 Resume Keyword Analysis
- 💡 Resume Improvement Suggestions
- 📈 Section-wise Feedback
- ⚡ Fast Resume Parsing
- 🎯 Job Application Optimization

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- PDF Parser
- Mammoth (DOCX Parser)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/smart-resume-ats-analyzer.git
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
